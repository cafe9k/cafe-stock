# CafeStock 技术文档索引

欢迎查阅 CafeStock 项目的技术文档。本文档提供了项目架构、核心类和功能流程的详细说明。

## 📚 文档列表

### 1. [技术架构文档](./architecture.md)

**推荐首先阅读**

详细说明项目的整体架构设计，包括：

- **整体架构**: 分层架构、设计模式
- **核心模块说明**: 主进程、数据库、服务层、IPC 通信等
- **渲染进程架构**: React 组件、Hooks、服务层
- **关键技术特性**: 性能优化、安全性、可维护性
- **扩展性设计**: 如何添加新功能模块

**适合人群**: 
- 新加入项目的开发者
- 需要了解整体架构的开发者
- 需要扩展功能的开发者

### 2. [核心类功能对照表](./core-classes.md)

**快速参考手册**

提供所有核心类的快速参考，包括：

- **类名和文件路径**: 快速定位代码位置
- **功能说明**: 每个类的职责和作用
- **关键方法**: 主要 API 和方法列表
- **数据库表结构**: 数据模型说明

**适合人群**:
- 需要查找特定功能的开发者
- 需要了解 API 的开发者
- 代码审查和重构时参考

**使用方式**:
- 使用 Ctrl+F 搜索类名或功能关键词
- 按模块分类查找相关类
- 查看关键方法了解使用方式

### 3. [核心功能流程图](./flowcharts.md)

**可视化流程说明**

使用 Mermaid 语法绘制的核心功能流程图，包括：

- **应用启动流程**: 从启动到就绪的完整流程
- **数据同步流程**: 公告、股票、股东数据同步
- **IPC 通信流程**: 主进程与渲染进程通信
- **自动更新流程**: 应用更新机制
- **数据处理流程**: 查询、聚合、分类等

**适合人群**:
- 需要理解功能执行流程的开发者
- 调试问题时追踪执行路径
- 学习项目数据流和业务逻辑

**查看方式**:
- 在支持 Mermaid 的 Markdown 查看器中查看（如 VS Code、GitHub）
- 或使用在线 Mermaid 编辑器查看

## 🎯 快速开始

### 对于新开发者

1. **第一步**: 阅读 [技术架构文档](./architecture.md) 了解整体架构
2. **第二步**: 查看 [核心功能流程图](./flowcharts.md) 理解主要功能流程
3. **第三步**: 使用 [核心类功能对照表](./core-classes.md) 查找需要的类和方法

### 对于特定任务

- **添加新功能**: 参考架构文档的"扩展性设计"章节
- **调试问题**: 查看流程图了解执行路径，使用对照表查找相关类
- **代码审查**: 使用对照表快速了解类的职责和方法
