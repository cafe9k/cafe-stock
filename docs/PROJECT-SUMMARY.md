## 📝 使用流程

### 开发者需要做的:

#### 第一步: 配置 Supabase
1. 访问 https://supabase.com
2. 创建新项目
3. 获取 API 凭证
4. 填入 `.env.local`

#### 第二步: 初始化数据库
1. 打开 Supabase SQL Editor
2. 运行 `database/init.sql`
3. 验证表创建成功

#### 第三步: 启动应用
```bash
npm run dev
```

#### 第四步: 开始使用
- 查看连接状态
- 添加库存数据
- 管理库存信息
